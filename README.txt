The format in which the questions should be stored in "questions.txt"
---------------------------------------------------------------------
Question1(Type the entire question)
Option1(Type the first option)
Option2(Type the second option)
Correct Option(1 or 2)
---------------------------------------------------------------------
The same can be done for any number of questions.
